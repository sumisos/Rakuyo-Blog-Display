「开发」树莓派配置Wi-Fi并安装基本开发环境

发表于 2018-11-25 更新于 2020-03-05 分类于 开发
本文字数： 1.9k 阅读时长 ≈ 4 分钟

准备工作

流量足够并且能开热点的手机 有路由器当然更好
以root用户登录的树莓派

注意：以下所有指令都没有sudo前缀，是因为我已经有root权限了

狗狗网线 配置wifi

  1. 执行iwlist wlan0 scan扫描附近的无线网

    确认你的网卡正常工作，回显的ESSID就是wifi名字

  2. 执行vim /etc/wpa_supplicant/wpa_supplicant.conf编辑wifi配置

  3. 按i键进入插入模式，在文件末尾加上

    然后ESC执行:wq保存退出

  4. 执行/etc/init.d/networking restart重启网络服务

  5. 执行reboot重启树莓派系统

  6. 此时putty的会话应该会断开，去看你的wifi连上了没

  7. 连上了就在电脑上再用cmd运行一次arp -a，MAC地址b8开头的类型为动态的IP在吗

  8. 用putty重连树莓派新的IP

    连网线的以太网 跟 连wifi的无线网 分配IP肯定有变化，应该只有前两个段一样

  9. putty会话显示login as:让你登录，成功连接，wifi配置完成

  10. 登录进去，使用ifconfig wlan0可以查看新的IP

安装开发环境

配置时区

  1. 执行date查看系统时间，如果慢了8个小时，说明是GMT时区
  2. 执行dpkg-reconfigure tzdata，选择Asia/Shanghai切换到CST（中国时区）

更换软件源

  1. 执行vim /etc/apt/sources.list，在亮着的第一行前面加#注释掉，在下面一行加入 :wq保存退出
  2. 执行apt-get update更新软件源
  3. 执行apt-get upgrade
  4. 执行apt-get dist-upgrade -y

安装LNMP环境

  1. 执行apt-get install nginx php7.0-fpm php7.0-cli php7.0-curl php7.0-gd php7.0-mcrypt php7.0-cgi安装nginx和PHP
  2. 执行service nginx start启动nginx服务
  3. 执行service php7.0-fpm restart重启php
  4. 执行vim /etc/nginx/sites-available/default修改nginx配置
  5. 将 替换为 :wq保存退出
  6. 执行service nginx restart重启nginx服务

安装git

  1. 执行apt-get install git安装git
  2. 后略

设置默认使用Python3

  1. python查看版本，虽然python2跟python3都安好了的，但一般默认使用2.6
  2. rm /usr/bin/python删掉之前的python链接
  3. ln -s /usr/bin/python3.5 /usr/bin/python新建python链接
  4. python查看版本，应该是3.5了吧

我设置之后pip用不了了，重新安装pip

  1. wget https://bootstrap.pypa.io/get-pip.py --no-check-certificate
  2. python get-pip.py
  3. pip -V查看pip版本
