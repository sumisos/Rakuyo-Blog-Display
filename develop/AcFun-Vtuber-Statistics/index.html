「开发」AcFun Vtuber 数据统计

发表于 2020-03-25 更新于 2020-03-27 分类于 开发
本文字数： 332 阅读时长 ≈ 1 分钟

这个点子其实蛮早就想到了，当时还没自鸣得意超过两分钟就搜到完成度极高的「VTBs in bilibili!」，顿时索然无味。
当时稍微考虑了一下架构，感觉绝大部分工作量都在前端，并没有如何在意。
当然分布式爬虫的设计确实很棒，用户反哺服务器，「我为人人，人人为我」的理念不就是H@H 区块链的内核吗。

然而今天下午自己上手从零开始写，被挫败感按在地上摩擦。
纸上得来终觉浅，绝知此事要躬行。

我菜是一方面，另一方面主站的API是真的磨人。
刚开始搜到「AcFun API 官方文档」心里一阵窃喜，点开人傻了。

想想B站也许真的是大站里面除了豆瓣以外对爬虫最友好的站了。
不过核心资源——比如说「真实直播源」是做了反爬，但也只是一个代理池就能打掉的东西，我没做除了懒就是笨。

是时候练手小程序了
